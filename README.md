# Jinette Ramos Photography Website

This is a photography portfolio website with responsive viewport scaling.

View your app in AI Studio: https://ai.studio/apps/drive/1niLFX66Orx6kWZYjNDRO-N2qsqFUqnlj

## Features

- **Fixed Aspect Ratio Viewport Scaling**: The website uses a 16:9 (1920x1080) design that scales proportionally on all devices while maintaining the desktop layout. See [VIEWPORT_SCALING.md](VIEWPORT_SCALING.md) for details.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
